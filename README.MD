### Django Todo CRUD 

#### Project Setup
1) Start virtual environment 
    ~~~ bash 
    pipenv shell
    ~~~
2) Install Django  
    ~~~ bash
    pipenv install django
    ~~~    
3) Create new project called `todo`
    ~~~ bash
    django-admin startproject todo
    ~~~    
4) Change directory into `todo` and launch server
    ~~~ bash
    python manage.py startproject todo
    ~~~
